# 🔐 Simple Login Page

A modern and responsive login page built using HTML, CSS, and JavaScript. This project demonstrates front-end web development fundamentals, including form design, styling, animations, and basic client-side interactions.

## ✨ Features

- Responsive design
- Clean and modern UI
- Smooth animations and transitions
- User-friendly login form
- Built with pure HTML, CSS, and JavaScript
- Beginner-friendly code structure

## 🚀 Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
